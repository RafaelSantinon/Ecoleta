## Tecnologias

- NodeJS
- Nunjucks
- SQLite
- HTML
- CSS

## Pré-requisitos

- Instalar uma IDE (Visual Studio Code)
- Instalar o Node.JS

```bash
# Clone este repositório
$ git clone <https://github.com/RafaelSantinon/html-css-basico-treino.git>

# Instalar as dependências
npm install

#Iniciar o projetos
npm start
```
## Rodando a aplicação

- Abrir um navegador e acessar localhost:3000
